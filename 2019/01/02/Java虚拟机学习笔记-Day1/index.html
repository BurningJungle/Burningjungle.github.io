Java虚拟机学习笔记#Day1

#走进Java#
1.JDK：用于支持java程序开发的最小环境。
2.JDK包括：java程序设计语言、java虚拟机、javaAPI库。
3.JRE：支持java程序运行的标准环境。
4.JRE包括：javaAPI类库中javaSE API子集和java虚拟机
64位虚拟机：
缺点：
内存问题：由于指针膨胀和各种数据类型对齐补白的原因，运行再64位系统上的虚拟机通常要比普通32位机多消耗10%-30%
的内存。
据多个机构测试表明，64位的运行速度再各项测试中都低于32位机，两者大约有15%的差距。

##java虚拟机发展历程##
1991年4月，在绿色计划中启动，目的在于消费性电子产品上运行的程序架构，计划产品名：Oak。
这是java的前身。但由于市场原因并不成功
1995年随着互联网潮流的兴起，Oak迅速找到适合自己的发展市场定位并开始蜕变为java。
1995年5月23日，Oak正式更名为java，在SunWorld大会上正式发布了java1.0。
1996年1月23日，JDK1.0发布，这是第一个java语言的正式运行版环境，代表技术包括：java虚拟机、Applet、AWT等。
1996年4月，10个主要的操作系统供应商声明其产品中嵌入了Java技术。
1996年5月底。Sun公司举行第一届JavaOne大会。
1996年9月，8.3个网页应用了java技术来进行制作。
1997年2月9日，Sun公司发布JDK1.1,技术代表：JAR文件格式、JDBC、JavaBeans、RMI。
Java语法新增内部类(Inner class)、反射(Reflection)。
1998年12月4日，JDK1.2发布。Sun公司将其java体系技术拆分为3个方向：面向桌面应用开发的J2SE、面向企业级开发的J2EE、
面向手机等移动终端开发的J2ME。代表性的Java技术：EJB、Java Plug-in、Java IDL、Swing。且该版本中第一次内置了JIT编译器，
(JDK1.2中曾经并存过3种虚拟机：Classic VM、HotSpot VM、Exact VM)语言和API级别添加了collections集合类和Strictfp关键字。
2000年5月8日，JDK1.3发布。
2002年2月13日，JDK1.4发布。
2004年9月30日，JDK1.5发布。
2006年12月11日，JDK1.6发布。
2006年11月13日，在JavaOne大会上，Sun最终宣布将Java开源。
2009年2月19日，JDK1.7发布。
2009年4月20日，Oracle公司收购Sun公司。
